# Kategorije i vrste IoC-a
 
## Vrste
- Atomic IoC - nedjeljivi ili nepromjenivi tragovi
- Izvedeni(computed) - izdvajamo konkretan podatak iz hrpe podataka
- Bihevioralni ( behavioral ) - Neki skup pokazatelja koji definiraju ponasanje ( kombinacija atomskih i izvedenih indikatora na temelju neke logike )
 
## Kategorije
- E-mail indikatori - jedan od najcescih vektora napada ( adrese, naslovi, privitci i linkovi )
- Mreza - koristi se za command and control, isporuku malware-a... ( URL, Domena, IP adresa)
- Host-based - indikacoti su dobiveni provodzenjem analize napadnutog racunala ( Hash vrijednost datoteka ( HASH uzima sve elemente datoteke i izradzuje niz nekih znakova i uvijek daje istu vrijednost), registry, dll datoteke )
- Bihevioralni - specificna ponasanja ( Power Shell, udaljeno izvrsavanje naredni )
