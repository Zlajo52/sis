# Taktike, Tehnike i Procedure
 
-Uzorci aktivnosti i metode karakteristicne za odredzene ili pojedine napadace
 
**Napadaci mogu biti:**
1. Svakodnevni ( ljudi koji rade greske )
2. Profesionalni ( kriminalne gupe npr. )
 
Napadacke grupe se jos nazivaju i **APT (Advanced Persistent Threat)**
 
**Taktike** - nacin na koji se napadaci ponasaju tokom faza napada ( one daju odgovor na "zasto?" )
 
**Tehnike** - alati koje koriste ( daju odgovor na pitanje "Kako?")
 
**Procedure** - odredzeni niz koraka koji napadaci koriste kako bi ostvarili svoj cilj
 
**Indikatori ugrozenosti**
 
- odnose se na podatke koju pokazuju da je naš informacijski sustav bio meta NAPADA
- oni se fokusiraju na istrazivanje stanja sustava nakon napada
 
 
**Indikatori napada**
- oni se fokusiraju na istrazivanje stanja sustava tokom napada
- ako se napad dogodi obicno je znatno skuplji od ulaganja u sigurnost
